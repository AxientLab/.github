# Contributing

Most AxientLab repositories are private while the protocol and research work is under active development. We welcome concise, responsible feedback through the contact route at [axient.app/contact](https://axient.app/contact).

When a repository is opened for external contributions, its repository-specific guidance takes precedence. Please keep proposed changes factual, reproducible, and explicit about the difference between research evidence, reference code, audit status, and production status.

Do not submit credentials, personal data, or unredacted security findings through public issues.
