# Security policy

If you believe you have found a security issue affecting an AxientLab project, please use the private contact route at [axient.app/contact](https://axient.app/contact) and include enough detail to reproduce the issue safely.

Please do not disclose vulnerabilities, secrets, user data, or exploit instructions in public GitHub issues. We will acknowledge a responsible report and coordinate an appropriate disclosure process.

Research and reference repositories are not a statement of deployed or audited production infrastructure.
