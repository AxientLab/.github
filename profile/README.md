# Axient.

Axient. researches and builds reference financial-accounting and risk mechanisms for leveraged event markets.

## What we work on

- Protocol and market-structure research
- Reference accounting and risk implementations
- Research documentation and reproducibility materials
- Design systems and developer-facing tooling

## Research boundary

Our public materials distinguish research, reference-code, audit, and production status.
Synthetic research outputs are not product-performance estimates, APY forecasts, safety guarantees, or evidence of a live deployment. Axient. is not presenting public production access through this GitHub profile.
